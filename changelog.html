Releases & Roadmap

January 2027 (Planned)
Planned Enterprise Cloud
Enterprise Private Cloud & Multi-AZ Clusters

Launch of managed private cloud instances. Supports automatic multi-region clustering, live container migrations, and zero-downtime database replication systems.

Planned Security & Network
Advanced DDoS Shielding & Anycast DNS Network
