CloudHoste Releases

Product Changelog

Follow the latest updates, features, improvements, and roadmap timelines on our secure cloud platform.

July 2026
Added File Storage
File Storage is now in General Availability

File Storage is now fully available across our secure PAR region nodes.

  • Kubernetes users: Dynamically provision Persistent Volumes with ReadWriteMany access using standard drivers.
  • Non-Kubernetes users: Attach multiple CPU or GPU cloud instances to shared managed systems via Console, CLI, or Terraform.
Added IAM Security
Resource-level IAM policies for Secret & Key Managers

Fine-grained security conditions have been implemented at the individual resource level:

  • Restricted access control rules for Secret Manager folders.
  • Asymmetric keys level permissions mapping inside Key Manager.
  • Specific policies for groups, applications, and custom SSH keys in IAM.
Changed IAM Console
IAM Policy Creation Interface Redesign

Our console interface for generating custom IAM rules has been modernized. Users can select scopes, permission categories, and groups in an intuitive workspace layout, reducing configuration steps.

Added CPU Instances
Server Instance Templates GA

Create predefined, immutable templates detailing server types, boot images, volume configs, cloud-init scripts, and network maps to deploy identical clusters repeatedly in seconds.

Deprecated Containers
Serverless Containers v1beta1 Deprecation

API v1beta1 is marked as deprecated, replaced fully by production v1. Transition timelines and API endpoints support guides are available in our official developer portal docs.

Added Audit Trail
Generative APIs Tracking in Audit Trail

Log activities, prompt metrics, model triggers, and general access parameters for our Generative AI models, tracking security changes inside our centralized compliance logging dashboard.

Added Elastic Metal
Proxmox Backup Server 4 Integration

We have integrated Proxmox Backup Server version 4, built on Debian 13 (Trixie), globally across our Beryllium, Iridium, and Lithium metal node configurations.

Added Kubernetes
Kubernetes Engine v1.36 Released

Control plane and node images have been updated to v1.36. Core components upgrades include:

  • Calico network plugin upgraded to v3.32.0; Cilium to v1.19.5.
  • Scaleway Block Storage CSI driver updated to v0.3.5.
  • Feature gates enabled: HPAScaleToZero, CPUManagerPolicyAlphaOptions.
June 2026
Added Elastic Metal
Proxmox VE 9.0 Availability

Deploy Proxmox VE 9.0 (Debian 13 Trixie based) on bare-metal server packages with advanced virtualization controls and integrated hypervisor backups.

Added Generative APIs
GLM 5.2 Large Language Model Available

Access GLM 5.2 on our AI APIs, ranking as the leading open-weight model for agentic execution, structured reasoning, and coding assistance inside development environments.

Added VPC Network
VPC Peering in General Availability

Scaleway VPC Peering is now in General Availability, supporting fine-grained VPC ingress routing configurations to link isolated secure subnets seamlessly.

Added Serverless Jobs
Automatic Retries for Serverless Jobs

Configure automatic retry policies for serverless execution tasks. If a routine fails due to intermittent crashes or network timeouts, it will automatically rerun according to user thresholds.

Added Serverless Jobs
Multiple Cron Triggers Support

Attach multiple CRON triggers to a single serverless job. Supports dynamic command and argument overrides for executing different parameters from the same code container.

Added Apache Kafka
Multi-AZ clusters for Apache Kafka®

Deploy brokers across 2 separate Availability Zones (AZ) with automatic failover, data replication, and zero event loss guarantees during localized infrastructure outages.

Added FinOps Control
FinOps Cost Management API Beta

Query, filter, and track granular cloud spending per-resource with custom time resolution using our new FinOps API. Built to support enterprise cloud budgeting optimization.
