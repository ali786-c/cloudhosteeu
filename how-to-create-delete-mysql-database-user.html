How to Create or Delete a MySQL Database or User

Databases are designed to store essential details and information across a set of tables controlled by a Database Management System (DBMS). MySQL is one of the most popularly used Database Management Systems.

It is a free-to-use software that provides the stability, reliability, and security required for an effective database management system. This article provides a walkthrough process of creating or deleting a MySQL database or user.

How To Create A MySQL Database

Follow these steps to set up a new empty database:

Step 1
Access CloudHoste Panel

Log into your CloudHoste Panel Control Panel and navigate to the Databases section. You can type in the keyword 'database' in the search bar for quick navigation.

Step 2
Select MySQL Databases

Click on the MySQL Databases tool from the list of database features.

Step 3
Name Database

Give your new database a name (for example: site_db).

Step 4
Create and Finalize

Click on the Create Database button. Once successfully configured, a success notification will show, and your new database will appear in the existing databases list.

How To Delete A MySQL Database

If you ever need to remove a database from your server:

1Select Database

Navigate to the Existing Databases list in your control panel.

2Click Delete

Click the Delete button next to the database you wish to remove.

3Confirm Action

A confirmation popup box will ask for verification. Click YES to finalize. Your database and all its tables will be permanently deleted.

Creating a MySQL Database User

We create a MySQL database user to provide additional security and authentication for your website's database. The user is assigned a login username and password, which differs from the Windows login credentials and the CloudHoste Panel control panel credentials.

Before any external program or user accesses a database, it must present the registered database credentials to authenticate connections.
How To Create A MySQL Database User
1Access Tool

Login to your CloudHoste Panel account, go to Databases and click on MySQL Databases.

2Input Credentials

Scroll down to the New MySQL User section. Type in your desired Username and a strong Password, then repeat the password.

3Create User

Click the Create User button. Your new user account will be successfully created.

How To Delete A MySQL Database User

To delete an existing user, scroll down to the Existing MySQL Users section, locate the target username, and click the Delete button.

Define User's Privileges

Now that you have successfully created a new user account, it is vital to assign user privileges to determine the user's level of control over the database. As a website owner, you do not want to grant full database access to just anyone.

Assign Database User Privileges
1Open Tool

Login to your CloudHoste Panel and go to the MySQL Databases section.

2Grant Access Section

Scroll down to the Grant Access To Database section.

3Map User & DB

Select the desired User from the drop-down menu and match it with the specific Database you wish to link.

4Add to DB

Click the Add user to database button. The system will save the privileges configuration instantly.

Conclusion

By following the step-by-step processes outlined in this article, you have learned how to create and delete databases, add MySQL database users, and assign database privileges.

If you encountered any issues while setting up these databases or permissions, please contact our support team, and we will be happy to help you out immediately.
